Processes scripts and creates a copy.
-Removes comments
-Cuts down white space
-Replaces local variable names with a shorthand name
-Removes all instances of the 'dbg' scriptcmd
-Does some error checking:
--Checks for unclosed brackets
--Checks for unclosed paretheses
--Checks for unclosed quotes (single and double)

The in and out file directory parameters must end with a slash '\' otherwise
the program will not work properly.

This program *might* not work properly without a dll. The installer will provide the dll:
https://answers.microsoft.com/en-us/windows/forum/windows_8-hardware/where-do-i-place-missing-msvbvm50dll-once/5c9cb691-7387-460e-9eaa-b2b70c0a4f19
